## 行內程式碼與程式區塊

行內程式碼示例：`console.log('Hello World')`

程式區塊示例：

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}
/* hi I'm Ssuyu */

const a = 1

greet('Suyu');
```
